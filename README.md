# Email Spam Classifier

## What it does
A machine learning model that automatically detects whether 
an email is spam or not spam (ham) using NLP techniques 
and classification algorithms.

## Tech Stack
- Python
- Jupyter Notebook
- Scikit-learn
- NLTK / Pandas / NumPy
- TF-IDF Vectorizer

## How it Works
1. Email text is cleaned and preprocessed
2. TF-IDF converts text into numerical features
3. Classification model predicts spam or not spam
4. Model is evaluated using accuracy and confusion matrix

## Results
- Achieved high accuracy on test dataset
- Model successfully filters spam emails
